Label Description
<header> A <header> es un contenedor generalmente
para enlaces de navegación o contenido
introductorio que contiene encabezados.
<nav> A <nav>se utiliza para definir un bloque
de enlaces de navegación, como menús y tablas de
contenido. Es importante tener en cuenta que
<nav>se puede usar dentro del <header>
elemento pero también se puede usar solo.
